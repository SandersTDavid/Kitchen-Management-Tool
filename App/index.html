Student Project UoP - Message please read

This web app was produced for the purpose of a university software engineering project. Please only use this website for experimentation with your training subject. It is recommended that you use an up to date Google Chrome browser with at least a 10 inch screen for best viewing purposes.
Any information entered will not be stored for more than one month and will not be viewed or manipulated in any way acording to the GDPR rules. Please do not use any personal information when entering the forms.

How can your kitchen benefit

Using these tools, you could reduce the time it takes to fully train a chef on multiple sections in the kitchen and reduce the amount of training time and save on staff expenses.

Why use these training methods

This website provides useful tools to help new or inexperienced chefs to get to know their section in the kitchen.
